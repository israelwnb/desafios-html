Estrutura HTML

a
Cria um hyperlink para páginas web, arquivos, endereços de email, localizações na mesma página, ou qualquer outra coisa que uma URL possa endereçar
button
É um elemento interativo ativado por um usuário com um mouse, teclado, dedo, comando de voz, ou outra tecnologia assistiva. Uma vez ativado, ele realiza uma ação programável, tal como enviar um formulário ou abrir uma caixa de diálogo.
code
Mostra seu conteúdo estilizado de maneira a indicar que o texto é um fragmento curto de código de computador. Por padrão, o texto do conteúdo é mostrado usando a fonte monoespaçada do user agent.
dl
Representa uma lista de descrição. O elemento armazena uma lista de grupos de termos (especificados usando o elemento dt) e descrições (fornecidos por elementos ds). Usos comuns para este elemento são para implementar um glossário ou mostrar metadados (uma lista de pares chave-valor).
